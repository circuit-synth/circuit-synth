Pythonic Circuit Design

Open-source Python library that seamlessly integrates into your professional KiCad workflow. Design new circuits or enhance existing projects with clean, readable, and verifiable code.

from circuit_synth import *

# Simple 3.3V voltage regulator
@circuit(name="voltage_regulator")
def voltage_regulator():
    """3.3V voltage regulator with input/output caps"""
    
    # Create power nets
    VIN = Net('VIN')
    VOUT = Net('VOUT')
    GND = Net('GND')
    
    # Voltage regulator
    reg = Component(
        symbol="Regulator_Linear:AMS1117-3.3",
        ref="U",
        footprint="Package_TO_SOT_SMD:SOT-223-3_TabPin2"
    )
    
    # Input and output capacitors
    c_in = Component(symbol="Device:C", ref="C", value="10uF")
    c_out = Component(symbol="Device:C", ref="C", value="22uF")
    
    # Connect the circuit
    reg["VIN"] += VIN
    reg["VOUT"] += VOUT
    reg["GND"] += GND
    
    c_in[1] += VIN
    c_in[2] += GND
    c_out[1] += VOUT
    c_out[2] += GND

# Generate KiCad project
circuit = voltage_regulator()
circuit.generate_kicad_project("voltage_regulator")

Get Started in Minutes

Create a complete ESP32-C6 development board project with hierarchical design in under 2 minutes

1

Quick Setup

Install uv and create your project:

# Install uv (Python package manager)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Create new project
uv init my_circuit_project
cd my_circuit_project
2

Add Circuit-Synth

Install and setup project template:

# Add circuit-synth
uv add circuit-synth

# Setup complete project template
uv run cs-new-project
3

Generate Your Circuit

Run the ESP32-C6 example with modular subcircuits:

# Generate complete KiCad project  
uv run python circuit-synth/main.py

# Result: ESP32-C6 dev board with:
# - USB-C with CC resistors
# - 5V‚Üí3.3V power regulation  
# - Programming interface
# - Status LED control
